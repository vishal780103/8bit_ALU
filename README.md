
# 8-bit ALU

Designed and implemented an 8-bit Arithmetic Logic Unit (ALU) supporting 8 arithmetic and logical operations.
Developed RTL code in Verilog HDL using combinational logic and case statements.
Created a testbench and verified functionality through simulation in EDA Playground.
Gained practical experience in digital design, RTL development, and verification concepts.

## Features

Designed using Verilog HDL.   
Supports 8 different arithmetic and logical operations.
Generates carry output during arithmetic operations.
Implemented using combinational logic with an always block and case statement.  
Verified using a dedicated Verilog testbench.  
Simulated using EDA Playground and Icarus Verilog.
## Tools Used

 Verilog HDL
- EDA Playground
- Icarus Verilog Simulator
- GTKWave (for waveform analysis)
## Learning Outcomes

Through this project, I gained hands-on experience in:

Verilog HDL coding
Combinational circuit design
Case statement implementation
Arithmetic and logical operations
RTL design concepts
Testbench development and simulation
Digital VLSI design fundamentals
